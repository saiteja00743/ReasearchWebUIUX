# UI Design System

## Brand

- Name: ResearchHub
- Tagline: The easiest way for students to publish and showcase research, projects, and technical articles.
- Personality: credible, fast, student-first, public-proof-of-work oriented

## Color Tokens

| Token | Value | Use |
| --- | --- | --- |
| Background | `hsl(210 40% 98%)` | App shell |
| Foreground | `hsl(222 47% 11%)` | Primary text |
| Primary | `hsl(173 80% 32%)` | Actions, links, highlights |
| Accent | `hsl(38 92% 55%)` | Select emphasis |
| Border | `hsl(214 32% 91%)` | Dividers and cards |
| Muted | `hsl(210 30% 95%)` | Tags and quiet surfaces |

## Typography

- Hero: 56-64px, bold
- Page title: 36-40px, bold
- Card title: 18-20px, bold
- Body: 14-18px, 1.6 line height

## Components

- Buttons: 6px radius, icon plus label where helpful
- Cards: 8px radius max, used for repeated items and tools
- Inputs: 44px height, visible border, label-first forms
- Tags: compact rounded rectangles for metadata
- Navigation: sticky top bar with publish CTA

## UX Principles

- Make publication status obvious.
- Keep public research pages SEO-first and fast.
- Let students generate metadata with AI, but keep them in control.
- Make profile and publication links easy to copy into resumes and LinkedIn.
- Use analytics to motivate repeat publishing.
