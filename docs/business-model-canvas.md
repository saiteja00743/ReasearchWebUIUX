# Business Model Canvas

| Block | Strategy |
| --- | --- |
| Customer Segments | Students, project teams, universities, research clubs, recruiters, technical communities |
| Value Propositions | Free publication pages, permanent links, credible profiles, AI metadata, analytics, discovery |
| Channels | LinkedIn, SEO, campus ambassadors, universities, technical clubs, hackathons |
| Customer Relationships | Self-serve publishing, creator analytics, ambassador support, institutional partnerships |
| Revenue Streams | AdSense, premium memberships, featured publications, sponsored universities, institutional analytics |
| Key Activities | Publishing workflow, review moderation, SEO growth, AI generation, analytics, university sales |
| Key Resources | Platform, publication graph, author profiles, moderation tools, brand trust |
| Key Partners | Universities, Cloudinary, MongoDB Atlas, Redis provider, Google OAuth, student communities |
| Cost Structure | Hosting, storage, AI usage, moderation, growth programs, sales, support |

## Pricing Hypothesis

- Free: publish unlimited papers with standard analytics
- Premium student: advanced analytics, custom profile, featured portfolio sections
- Featured publication: fixed campaign pricing
- University sponsor: annual subscription for branded university page and analytics
